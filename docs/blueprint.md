# **App Name**: Dashboard Genérico DicipWare

## Core Features:

- Visualización de KPIs: Muestra de Indicadores Clave de Desempeño (KPIs) en varios formatos de gráfico (línea, barras, pastel).
- Filtrado por Rango de Fechas: Selección personalizada de rangos de fechas para filtrar los datos.
- Diseño Responsivo: Diseño adaptable a distintos tamaños de pantalla.
- Insights Inteligentes: Análisis impulsado por IA que detecta tendencias y patrones para resaltar hallazgos clave. La herramienta decide cuándo es más relevante incluirlos.
- Panel de Administrador: La pagina debe tener los siguientes items en el dashboard Panel de control, Totes, Usuarios - esto debe ser el panel de administrador

## Style Guidelines:

- Azul oscuro (#3F51B5) para una sensación profesional y confiable.
- Gris muy claro (#F5F5F5) para una apariencia limpia y moderna.
- Verde azulado (#009688) para resaltar métricas clave y elementos interactivos.
- 'Inter', sans-serif, para una legibilidad clara y una estética moderna.
- Diseño basado en cuadrícula para una organización escalable y ordenada del contenido.
- Uso consistente de íconos de diseño material para una navegación intuitiva.
